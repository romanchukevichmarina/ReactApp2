# Проект React js для НИСа

В приложении 4 страницы: главная ("/"), страница регистрации ("/signup"), страница авторизации ("login") и страница со списком мониторов ("index"). Страница со списком товаров приватная и доступна только после регистрации/авторизации. Товары загружаются по 10 штук при нажатии на кнопку, после того как данные заканчиваются, кнопка отключается. При ошибке подключения к серверу высвечивается alert с предупреждением.
